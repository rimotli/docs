Signup Endpoint

POST /api/auth/signup

endpoint for creating users on avocat mobile app

http://127.0.0.1:8888/api/auth/signup

{
    "username":"exampleusername",
    "email":"example@example.com",
    "password":"examplepassword000",
    "confirmPassword":"examplepassword000",
    "phone":"+23400000000",
    "userType":"client"
}
**NB userType can either be 'client' or 'lawyer'** <br/>
ParameterDescription
Usernameusername of the user
Emailemail of the user
Passwordpassword of the user
ConfirmPasswordconfirm password of the user
phonephone number of the user
userTypeuserType of the user

NB phone number from any country

  • phone prefix supported
PrefixCountry
+234Nigeria

signup resposne

STATUS 200
"a verification mail has been sent to example@example.com"

common errors when consuming the signup endpoint

  • when a username parameter is not provided
STATUS 400 Bad Request
"\"username\" is required"
  • when an email parameter is not provided
STATUS 400 Bad Request
"\"email\" is required"
  • when a password parameter is not provided
STATUS 400 Bad Request
"\"password\" is required"
  • when a confirm password parameter is not provided
STATUS 400 Bad Request
"\"confirmPassword\" is required"
